ライフゲームとは

ライフゲームは生命の誕生、進化、淘汰などのプロセスを簡易的なモデルで再現したシミュレーションゲームです。

単純なルールでその模様の変化を楽しむことができるパズル要素も含んでいます。


#
生命の誕生
隣接した８マスに生きているセルが三つある場合、次の世代でそのセルに生命が生まれます。


　　　-----          -----
      |oo |          |oo |
      |o  |	  →     |oo |
	  |   |          |   |
	  -----          -----

#
生命の生存
隣接した８マスに生きているセルが二つあるいは三つならば、そのセルは次の世代も存続します。

　　　------       ------
      |    |       |    |
	  | oo |   →　| oo |　
	  | oo |       | oo |
	  |    |       |    |
	  ------       ------


#
生命の死滅（１）
隣接した８マスに生きているセルが一つ以下のとき、次の世代でそのセルは過疎により死滅します

      
　　　-----          -----
      |   |          |   |
      |oo |	  →     |   |
	  |   |          |   |
	  -----          -----



#
生命の死滅（２）
隣接した８マスに生きているセルが四つ以上のとき、次の世代でそのセルは過密より死滅します。


　　　-----          -----
      |ooo|          |ooo|
      |oo |	  →     |o  |
	  |   |          |   |
	  -----          -----
